Welcome to IHNIL

IHNIL (I Hate Nested If Loops) is a small, command-line tool for use on target Python documents to evaluate and optimize "if" loops.

Why IHNIL

Nesting multiple if loops within one another, without additional aspects such as an elif or an else or additional lines of code between if's, can be costly, particularly if there are several instances of this and the file is particularly large. This project looks to provide assistance in correcting this error.
